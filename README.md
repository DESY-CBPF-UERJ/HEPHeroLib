# ![HEPHeroLib](logo.svg)

**HEPHeroLib - Library used to process data from the HEPHero framework**

# Installation

Download from Git

```bash
git clone https://github.com/DESY-CBPF-UERJ/HEPHeroLib.git
```

Update package

```bash
cd HEPHeroLib/
bash update.sh
```

See the version

```bash
cat src/__init__.py
```
